Instructions for users:

1) Select the dataset.
There are 9 available datasets of RNA-seq measurements:
- ENCODE (human + mouse model organisms)
- Bgee (human + mouse)
- GTEx (human)
- Bluprint
- Roadmap Epigenenomics
- FlyBase (drosophila)


2) It is not necessery but you can also donload your own file.
This is a nice example of how a dataset file should look like

3)Check correlation table. The generated table will have 3 colomns.
Column 1 - name of experiments
Column 2 - correlation coefficients
Column 3 - scaled correlations
#feel free to download the table in txt format at the bottom of the page.


4)Static Heatmap.
The static heatmap tab will display a clustered heatmap representation
of the correlation matrix: each row and column represent a single
experiment.
The colour legend can be found below the heatmap: white represents
a correlation coefficient equal to 0, red equals to 0.5, and black equals t


5)Responisve Heatmap. This tab displays an interactive plot provided through the plot.ly API.
I It represents the same version of the Static heatmap presented in the previous section,
without the side dendrograms (trees).

6)The sample metadata tab will display metadata information (experiment name, cell type,
url of the original data, etc.) for the selected dataset.
